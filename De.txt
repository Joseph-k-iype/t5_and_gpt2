## Hard constraints (re-read before writing any line)

- Every write endpoint: OPAService.check_access() BEFORE any FalkorDB write. No exceptions.
- FalkorDB calls: always `await asyncio.to_thread(graph.query, cypher, params)`. Never call sync client from async handler.
- IDs: `from ulid import ULID; str(ULID())` everywhere.
- Pydantic v2: `model_config = ConfigDict(from_attributes=True)`. No v1 syntax.
- React: no try/catch in components. All errors via TanStack Query `error` state → `<ErrorBanner error={error} />`.
- AntV G6: double-lazy — ExplorePage is React.lazy() in App.tsx, ExploreView is React.lazy() inside ExplorePage. Never import G6 in the main bundle.
- Tailwind only. No inline styles, no CSS modules.
- All icons from lucide-react.
- All animations: Framer Motion for React state-driven transitions, GSAP only for DOM refs (sidebar, canvas).
- Naming: Python snake_case files/functions, TypeScript kebab-case files, PascalCase components.
